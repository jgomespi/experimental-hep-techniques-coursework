# Data

This directory contains the small text-based experimental datasets used by the notebooks.

## Subdirectories

- `gamma_spectroscopy/`: spectra and source measurements.
- `geiger_muller/`: plateau, source, background, and dead-time measurements.
- `scintillator_pmt/`: scintillator and PMT characterization data.
- `coincidence/`: timing coincidence measurements.
- `adc/`: pedestal, charge, and signal-output data.
