# Notebooks

This directory contains the organized analysis notebooks from the course.

## Contents

- `01_signal_transmission_and_impedance.ipynb`: signal propagation and impedance-matching studies.
- `02_gamma_spectroscopy_calibration.ipynb`: gamma spectroscopy and energy calibration.
- `03_geiger_muller_plateau_dead_time.ipynb`: Geiger-Muller plateau, source counting, background, and dead-time analysis.
- `04_scintillator_pmt_characterization.ipynb`: scintillator and photomultiplier characterization.
- `05_coincidence_timing.ipynb`: timing and coincidence measurements.
- `06_adc_signal_pedestal_analysis.ipynb`: ADC pedestal, charge, and signal analysis.
